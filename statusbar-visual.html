Bottom bar · download-details redesigns

Five directions for the window's bottom status bar. Today it shows only the speed-limit snail, ↓/↑ totals, and the profile picker — none of the actual download details (how many are running, overall progress, ETA, session totals). Each concept adds that, in a different visual key. Speeds & the progress bar animate live.

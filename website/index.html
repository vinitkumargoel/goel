Network aggregator

Wi-Fi + Ethernet + hotspot. One download.

Something rare: Goel° splits a single download across every network adapter you have — by byte range — and merges the pieces on disk. Plug in a cable mid-download and watch the combined number climb.

  • 1–8 streams per adapter, each with its own cap
  • VPN-safety guards keep traffic where it belongs
  • Try it: click an adapter to unplug it — the rest pick up the slack →

FAQ

Quick answers.

Do I need Homebrew or any libraries?

No. Every native library — libtorrent, libssh2, libcurl, OpenSSL — is bundled inside the app. Download the .dmg, drag to Applications, done.

Which Macs does it run on?

macOS 14 (Sonoma) or later, Apple Silicon and Intel. A notarized release just opens; for an un-notarized build — a beta, say — right-click the app → Open the first time.

Is it free?

Yes — Goel° is open source under the MIT license.

Can it download from video sites?

Official releases bundle yt-dlp: the “Resolve Media” button turns a video-site page into a direct download. HLS streams save to a clean .mp4.

Does it really run on Linux?

Yes — the same engine ships as GoelDaemon, a headless service driven entirely from the built-in web portal. Prebuilt for x86_64; build from source for ARM64.

Is remote access safe?

The portal is off by default. When on, it's token- or password-authenticated, supports read-only mode and session timeouts, and refuses to expose over the LAN without a password.

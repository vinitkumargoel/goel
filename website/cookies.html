Cookie Policy

Last updated: 21 July 2026

Short version: this site sets no cookies. There is nothing to accept, so there is no cookie banner.

No cookies, no trackers

This website is a static page. It sets no first-party cookies, no third-party cookies, no advertising or analytics identifiers, and it does not use localStorage or similar browser storage to track you.

Third-party requests

The site loads its typefaces from Google Fonts. That request delivers font files only — it does not set cookies in modern browsers — but Google's servers do see your IP address and user agent, governed by Google's privacy policy.

Links to GitHub (source code, releases) take you to GitHub, which manages its own cookies under its own policy.

The Goel° app

The desktop app and Goel° Web are not websites you visit — they run on your own machine. Goel° Web uses a session token to keep you signed in to your own portal; that stays between your browser and your machine and is not shared with anyone.

Changes

If this ever changes — for example, if we add privacy-respecting analytics — this page will be updated first, with a new date.
